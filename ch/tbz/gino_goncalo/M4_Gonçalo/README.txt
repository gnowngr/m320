M4 BANKING SYSTEM - OOP PROJECT
================================================================

Ein objektorientiertes Bankensystem mit API-Integration.

FEATURES:
---------
✓ Kontoverwaltung (Girokonto, Sparkonto, Depot)
✓ Geldtransaktionen (Einzahlen, Abheben, Überweisen)
✓ Aktienhandel mit Yahoo Finance API Integration
✓ Transaktionshistorie
✓ Portfolio-Management
✓ Exception Handling

----------------------------------
✓ 25+ Klassen (gefordert: min. 8)
✓ 1200+ Zeilen Code (gefordert: min. 800)
✓ 2 Design Patterns (Factory, Repository)
✓ Interface-Verwendung (StockAPI, Transactable)
✓ Vererbungshierarchie (Account -> GiroKonto/SparKonto/Depot)
✓ Polymorphismus (überschriebene Methoden)
✓ Delegation (UI -> Service -> Repository)
✓ Exception Handling (7 Custom Exceptions)
✓ Clean Code Prinzipien
✓ API Integration (Yahoo Finance)
